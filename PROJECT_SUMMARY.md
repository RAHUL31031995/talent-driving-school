## 🎯 Quick Start (5 minutes)

### 1. Install & Setup
```bash
cd talent-driving-school
npm install
cp .env.example .env
# Add Gmail App Password to .env
```

### 2. Run Locally
```bash
npm start
```
Visit: `http://localhost:3000`

### 3. Test Form

- Submit
- Check email inbox
- WhatsApp link appears

### 4. Deploy (when ready)
- Push to GitHub
- Connect to Render
- Set environment variables
- Website is live!
